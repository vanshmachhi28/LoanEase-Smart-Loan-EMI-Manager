# 💰 LoanEase – Smart Loan & EMI Manager

A modern Django-based Loan Management and EMI Calculator System designed to simplify loan processing, EMI calculation, repayment tracking, and customer management for financial institutions.

---

# 🚀 Features

✅ Secure User Authentication  
✅ Loan Application System  
✅ EMI Calculator with Charts  
✅ PDF EMI Report Generation  
✅ Loan Approval & Rejection  
✅ Customer Dashboard  
✅ Admin Dashboard  
✅ Loan Verification System  
✅ Interest & Penalty Management  
✅ Document Upload System  
✅ Responsive Bootstrap UI

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/vanshmachhi28/LoanEase-Smart-Loan-EMI-Manager.git
```

---

## Create Virtual Environment

```bash
python -m venv venv
```

---

## Activate Environment

### Windows

```bash
venv\Scripts\activate
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Configure Database

Create MySQL Database:

```sql
CREATE DATABASE loanease_db;
```

Update credentials in:

```bash
LoanEase/settings.py
```

---

## Run Migrations

```bash
python manage.py makemigrations
python manage.py migrate
```

---

## Start Server

```bash
python manage.py runserver
```
